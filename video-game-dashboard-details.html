VGS Dataset Analysis

Introduction

The VGS Dataset provides insights into various video games' sales, ratings, and performance across multiple platforms. This project focuses on analyzing key patterns and trends within the dataset to derive meaningful insights.

Objectives

  • Identify top-performing games based on sales.
  • Analyze platform performance over time.
  • Explore regional sales distribution.
  • Present insights through pivot charts for clarity.

Methodology

  • Data cleaning and preparation using Microsoft Excel.
  • Creation of pivot tables for organized data analysis.
  • Visualization through pivot charts for easy interpretation.

Data Preparation

  • Handled missing values and duplicates.
  • Formatted data columns appropriately.
  • Created additional columns for better categorization.
  • Validated data accuracy before visualization.

Data Analysis

  • Analyzed sales data by genre, region, and platform.
  • Identified top-selling games globally.
  • Compared sales performance between regions.

Data Visualization

  • Created pivot charts to show top-selling games.
  • Developed charts to depict regional performance trends.
  • Visualized platform-based sales comparisons.

Key Findings

  • Action and sports genres dominate global sales.
  • North America leads in video game sales compared to other regions.
  • Specific platforms like PS4 and X360 consistently performed better.
